# Как обновить zapret
Инструкция подойдет не всем !!!

+ Установка терминала если его нет [Terminal](https://github.com/Skardag/zapret/blob/main/%D0%A3%D1%81%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BA%D0%B0%20%D1%82%D0%B5%D1%80%D0%BC%D0%B8%D0%BD%D0%B0%D0%BB%D0%B0.pdf)
+ Установка Zapret [Zapret](https://github.com/Skardag/zapret/blob/main/%D0%A3%D1%81%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BA%D0%B0%20zapret%20%D0%B8%D0%B7%20%D1%82%D0%B5%D1%80%D0%BC%D0%B8%D0%BD%D0%B0%D0%BB%D0%B0.pdf)
+ Обновление Zapret [Zapret](https://github.com/Skardag/zapret/blob/main/%D0%9E%D0%B1%D0%BD%D0%BE%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5%20Zapret%20%D0%B8%20%D0%9F%D0%BE%D0%B4%D0%B1%D0%BE%D1%80%20%D1%81%D1%82%D1%80%D0%B0%D1%82%D0%B5%D0%B3%D0%B8%D0%B8.pdf)
+ Дополнительно у кого не заработал установить [DNS over HTTPS](https://github.com/Skardag/zapret/blob/main/%D0%A3%D1%81%D1%82%D0%B0%D0%BD%D0%BE%D0%B2%D0%BA%D0%B0%20DNS%20over%20HTTPS.pdf)

1. Зайдите на веб интерфейс

+ 4C Вход [192.168.31.1](http://192.168.31.1)
+ AX3000T Вход [192.168.32.1](http://192.168.32.1)

2. Вход в Веб-интерфейс\
Логин: root\
Пароль: 12345 / Либо пустой

3. Зайдите в раздел Служба -> Терминал (Terminal) !!! Если нет терминала в конце найдете как ее установить!
4. Повторяете вход пункт -> 2 (При вводе пароля не видно набора символов это нормально!)

5. Останавливаем zapret
```
/etc/init.d/zapret stop
```

6. Запуск Zapret-Manager (Готовый скрипт для обновления и установки)
```
sh <(wget -O - https://raw.githubusercontent.com/StressOzz/Zapret-Manager/main/Zapret-Manager.sh)
```

ВНИМАНИЕ СКРИПТ БЫВАЕТ ЧТО ОБНОВЛЯЕТСЯ И ПУНКТЫ ТАКЖЕ МОГУТ МЕНЯТСЯ МЕСТАМИ ИЛИ ДОБАВЛЯЕТСЯ НОВЫЕ!!!

Выбираете нужный пункт смотря что вам нужно 

Если нужно просто обновить то можете выбрать "8" пункт он автоматически переустановить на новую версию

Если не работает ютуб выберите\
-- 2 Пункт ( Меню стратегий )\
-- 4 Пункт ( Подобрать стратегию для YouTube )\
---- Дальше подбираете свой вариант Если все подходит то жмем Enter ( Применить настройки ) S - Оставить N - Продолжить подбор



Если нет терминала 

1. Заходим раздел "Система (System)"
2. Пакеты (Software)
3. Обновить список (Update list) - Зеленая кнопка
4. В поиске набираем "Terminal"
5. Находим внизу списка "luci-app-ttyd" и устанавливаем справа будет кнопка
6. После установки нужно пере зайти и можно переходить к zapretу
